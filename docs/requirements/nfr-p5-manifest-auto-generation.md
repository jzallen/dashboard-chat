# NFR-P5: Manifest Auto-Generation

## Tag

P5 — Preview: Usability

## Invariant

> The system SHALL auto-generate a semantic manifest from project Views and Reports. Users SHALL NOT manually assemble manifests.

## Status

**Not implemented** — tracked in `planner-docker-integration` proposal

## Verification Method

Verify that creating Views and Reports in a project results in an automatically generated semantic manifest without any manual assembly steps.
